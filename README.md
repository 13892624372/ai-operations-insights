<h2>三、核心发现</h2>

<h3>发现1：杭州出现“高薪”岗位</h3>
<p>本次样本中，杭州出现了25-40K和20-35K的高薪岗位，薪资上限甚至超过北京和上海。</p>

<h3>发现2：大厂提供更高薪资上限</h3>
<p>大厂岗位的薪资上限普遍在25-40K，而中小企业集中在12-22K区间。</p>

<h2>四、结论</h2>
